Bem-vindo!

Este é um site criado para revisar os conteúdos principais estudados na unidade curricular Publicar Aplicações Web.

HTML

HTML é a estrutura básica da página. Aqui aprendemos sobre tags, semântica, acessibilidade e organização dos elementos.

CSS

CSS define a aparência do site. Revisão de seletores, classes, IDs, responsividade e layout moderno com Flexbox e Grid.

JavaScript

JavaScript adiciona dinamismo. Aqui utilizamos eventos, funções, manipulação do DOM e interatividade simples.

Publicação (Deploy)

Revisão do fluxo de publicação usando GitHub e Vercel. Versionamento, commits, push e deploy automático.
